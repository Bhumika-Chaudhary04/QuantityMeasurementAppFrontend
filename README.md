# 📏 Quantity Measurement App – Frontend

The **Quantity Measurement App Frontend** is a modern React-based web application designed to provide an intuitive and interactive interface for performing unit conversions, arithmetic operations on measurable quantities, comparisons, and tracking operation history.

This frontend is part of the complete **Quantity Measurement App ecosystem** and integrates seamlessly with a Spring Boot backend.

---

## 🚀 Overview

The application allows users to work with different measurable quantities such as length, temperature, and more. It provides a clean dashboard where users can:

- Perform **unit conversions** across multiple measurement types
- Execute **arithmetic operations** such as addition, subtraction, multiplication, and division
- Compare two quantities
- Track and manage **operation history**
- Clear history when needed
- Access **login and signup pages** (UI ready)
- Enjoy a **responsive and user-friendly interface**

---

## 🛠 Tech Stack

- **React**

- **JavaScript (ES6+)**
